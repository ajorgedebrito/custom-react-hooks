## useUpdateLogger

> Este **Hook** serve para fazeres console.log(value) sempre que alteras o estado da prop:**value**

**How to use**
```bash
useUpdateLogger(value);
```
|                |Props                          |Type                         |
|----------------|-------------------------------|-----------------------------|
|value           |`'Isn't this fun?'`            |unknown                      |
